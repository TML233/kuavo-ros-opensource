# 轮臂手柄控制说明

## 概述

本文档说明如何使用手柄控制轮臂机器人，包括三种控制模式及其切换方式。

目前支持两种手柄：
1. **BT2Pro / BT2 手柄**：通过USB连接，使用标准 `joy_node` 驱动
2. **G12遥控器**：通过SBUS接收器（CP2102串口）连接，使用 `h12pro_controller_node` 驱动

## 控制模式

系统提供三种控制模式：

1. **cmd_vel 模式**
2. **cmd_vel_world 模式**
3. **躯干控制模式**（cmd_lb_torso_pose）

---

## 一、BT2Pro / BT2 手柄

在使用终端命令启动机器人时，请务必先确认接收器已安装到位，并选好对应的手柄类型，最后再执行启动。  
否则，系统可能会出现按键映射错误（比如BT2手柄被识别成BT2 Pro），导致无法通过按键正确控制机器人的行为情况。  
这是因为目前的BT手柄在5W平台上还不支持动态检测和自动重载映射文件，所以必须在启动前就确定好配置。


```bash
roslaunch humanoid_controllers load_kuavo_real_wheel.launch joystick_type:=bt2
roslaunch humanoid_controllers load_kuavo_real_wheel.launch joystick_type:=bt2pro

```

### 模式切换

- **LB + A键（BUTTON_STANCE）**：切换到 `cmd_vel` 模式
- **LB + Y键（BUTTON_WALK）**：切换到 `cmd_vel_world` 模式
- **LB + B键（BUTTON_TROT）**：切换到 `躯干控制`模式

### 默认模式

系统启动时默认模式为 **cmd_vel** 模式。

### 其他功能按键

- **BACK键**：杀掉程序

### 模式详细说明

#### 1. cmd_vel 模式 / cmd_vel_world 模式

**话题**：`/cmd_vel` 或 `/cmd_vel_world`

**控制方式**：
- **左摇杆上下方向（X轴）**：控制 `linear.x`（前进/后退线速度）
- **左摇杆左右方向（Y轴）**：控制 `linear.y`（左右移动线速度）
- **右摇杆左右方向（Yaw轴）**：控制 `angular.z`（绕Z轴旋转角速度）

**死区**：0.05

#### 2. 躯干控制模式（cmd_lb_torso_pose）

**话题**：`/cmd_lb_torso_pose`

**控制方式**：

此模式下需要配合按键使用摇杆进行控制。

##### 躯干X位置控制
- **LB + 左摇杆上下方向**：控制 `linear.x`（躯干X位置）
  - **注意**：需要按住LB键才触发控制，松开LB键时保持当前值

##### 躯干Z位置控制
- **LB + 右摇杆上下方向**：控制 `linear.z`（躯干Z位置）
  - **注意**：需要按住LB键才触发控制，松开LB键时保持当前值

##### 躯干Yaw角度控制
- **RB + 右摇杆左右方向**：控制 `angular.z`（躯干Yaw角度）
  - **注意**：需要按住RB键才触发控制，松开RB键时保持当前值

##### 躯干Pitch角度控制
- **RB + 左摇杆上下方向**：控制 `angular.y`（躯干Pitch角度）
  - **注意**：需要按住RB键才触发控制，松开RB键时保持当前值

##### 复位功能
- **RT键**：按下RT键时，所有躯干控制量清零（复位到初始位置）

**死区**：0.05

**特殊说明**：
- 进入躯干控制模式时，系统会自动切换到MPC的ArmOnly模式
- 退出躯干控制模式时，系统会恢复之前的MPC模式
- 所有控制值采用积分方式：摇杆为正则增加，为0则保持不变，为负则减小

### 使用提示（BT2Pro）

1. **模式切换**：模式切换前要保证手柄控制量为0，且保持2秒钟，之后才能进行模式切换
2. **紧急停止**：按下BACK键可以杀死程序
3. **死区处理**：摇杆输入值小于0.05时会被视为0，避免微小抖动

## 二、G12遥控器

### 硬件配置

- **接收器**：CP2102 USB to UART Bridge Controller
- **串口设备**：`/dev/usb_remote`（通过udev规则绑定）
- **波特率**：100000（SBUS协议）
- **通道数**：12通道

### 通道布局

| 通道 | 功能 | 说明 |
|------|------|------|
| CH1 | 右摇杆左右 | Yaw |
| CH2 | 右摇杆上下 | Z轴 |
| CH3 | 左摇杆上下 | X轴 |
| CH4 | 左摇杆左右 | Y轴 |
| CH5 | E开关 | 三段开关 |
| CH6 | F开关 | 三段开关 |
| CH7 | A按钮 | 自复位 |
| CH8 | B按钮 | 自复位 |
| CH9 | C按钮 | 自复位 |
| CH10 | D按钮 | 自复位 |
| CH11 | G拨杆 | 自回中拨轮 |
| CH12 | H拨杆 | 自回中拨轮 |

### 安全开关

- **E开关和F开关**必须同时处于**中间位置**时，所有控制指令才生效（切换模式等指令）
- 任一开关不在中间位置时，机器人停止响应手柄输入（走路不会影响）

### 模式切换

- **G极值 + A按钮**：切换到 `cmd_vel` 模式
- **G极值 + B按钮**：切换到 `cmd_vel_world` 模式
- **G极值 + C按钮**：切换到 `躯干控制`模式

> "G极值"指G拨杆拨到任意方向的极限位置（接近282或1722），松手后自动回中。

### 默认模式

系统启动时默认模式为 **cmd_vel** 模式。

### 紧急停止

- **C + D 同时长按1秒**：触发紧急停止（杀死程序）

### 模式详细说明

#### 1. cmd_vel 模式 / cmd_vel_world 模式

**话题**：`/cmd_vel` 或 `/cmd_vel_world`

**控制方式**：
- **左摇杆上下**：控制 `linear.x`（前进/后退线速度）
- **左摇杆左右**：控制 `linear.y`（左右移动线速度）
- **右摇杆左右**：控制 `angular.z`（绕Z轴旋转角速度）

**死区**：0.05

#### 2. 躯干控制模式（cmd_lb_torso_pose）

**话题**：`/cmd_lb_torso_pose`

**控制方式**：

此模式下通过G/H拨杆配合摇杆进行控制。进入躯干控制模式时，底盘自动锁定（不会移动）。

##### 躯干X位置控制
- **G极值 + 左摇杆上下**：控制 `linear.x`（躯干X轴前后位置）
  - **注意**：需要拨住G拨杆才触发控制，松手后保持当前值

##### 躯干Z位置控制
- **G极值 + 右摇杆上下**：控制 `linear.z`（躯干Z轴上下位置）
  - **注意**：需要拨住G拨杆才触发控制，松手后保持当前值

##### 躯干Yaw角度控制
- **H极值 + 右摇杆左右**：控制 `angular.z`（躯干Yaw航向角度）
  - **注意**：需要拨住H拨杆才触发控制，松手后保持当前值

##### 躯干Pitch角度控制
- **H极值 + 左摇杆上下**：控制 `angular.y`（躯干Pitch俯仰角度）
  - **注意**：需要拨住H拨杆才触发控制，松手后保持当前值

##### 躯干复位
- **G + H 同时极值保持2秒**：所有躯干控制量清零（复位到初始位置）

**死区**：0.05

**特殊说明**：
- 进入躯干控制模式时，系统会自动切换到MPC的ArmOnly模式，并禁用底盘速度控制节点
- 退出躯干控制模式时，系统会恢复之前的MPC模式，并恢复底盘速度控制节点
- 所有控制值采用积分方式：摇杆为正则增加，为0则保持不变，为负则减小
- G/H拨杆为自回中拨轮，只有在拨到极值位置时才激活控制，松手后自动回中并保持当前值

### 使用提示（G12）

1. **安全开关**：使用前确保E和F开关都在中间位置，否则所有控制无效
2. **模式切换**：将G拨杆拨到极值位置后按下A/B/C按钮进行模式切换
3. **紧急停止**：同时按住C和D按钮1秒触发紧急停止
4. **躯干控制**：拨住G或H拨杆配合摇杆调整躯干位置，松手保持当前值
5. **躯干复位**：同时将G和H拨到极值位置保持2秒，躯干回到初始位置
6. **死区处理**：摇杆输入值小于0.05时会被视为0，避免微小抖动

## 启动方式

### BT2Pro 手柄

```bash
roslaunch humanoid_controllers load_kuavo_gazebo_sim_wheel.launch joystick_type:=bt2pro
# 或
roslaunch humanoid_controllers load_kuavo_real_wheel.launch joystick_type:=bt2pro
```

### G12 遥控器

```bash
roslaunch humanoid_controllers load_kuavo_gazebo_sim_wheel.launch joystick_type:=h12
# 或
roslaunch humanoid_controllers load_kuavo_real_wheel.launch joystick_type:=h12
```
